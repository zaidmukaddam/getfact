Get a random or all fact from the database that you store

You need API Key to access your data

Get your fact

Get a random fact or all facts related to your API Key

Method: GET

Path: https://getfact.deno.dev/api/v1/fact

Parameters

Parameter Required type Default Description
api_key Yes string - Your API Key
single No boolean true If true return 1 random fact, else return all facts
ignore No string - only work if single is true. Prevent getting the same fact

Example:

  • Get one random fact

    https://getfact.deno.dev/api/v1/fact?api_key=[API_KEY]

  • Get one random fact and ignore fact

    https://getfact.deno.dev/api/v1/fact?api_key=[API_KEY]&ignore=[FACT_TO_IGNORE]

  • Get all facts

    https://getfact.deno.dev/api/v1/fact?api_key=[API_KEY]&single=false

Add new fact

add new fact to the database related to your API key

Method: POST

Path: https://getfact.deno.dev/api/v1/fact

Parameters

Parameter Required type Default Description
api_key Yes string - Your API Key
fact Yes string - Fact to add to the database

Delete fact

delete fact related to your API key

Method: DELETE

Path: https://getfact.deno.dev/api/v1/fact

Parameters

Parameter Required type Default Description
api_key Yes string - Your API Key
fact Yes string - Fact to delete
